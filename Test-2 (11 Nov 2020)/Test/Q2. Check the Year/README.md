## 2. Check The Year

Given a number <i>N</i>, the task is to check if <i>N</i> is a leap year.

<br>
<br>
Sample Input 0:<br>

```
2000
```

<br>
<br>
Sample Output 0:<br>

```
yes
```

<br>
<br>
Sample Input 1:<br>

```
1997
```

<br>
<br>
Sample Output 1:<br>

```
no
```
