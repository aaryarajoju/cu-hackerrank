## Adding Two Numbers

Determine the integer floor sum of two floating point numbers. The floor is the turnacated float value, i.e anything after the decimal point is dropped.
<br>
<br>
<b>Example:</b><br>
<i>floor(1.1+3.89) = floor(4.99) = 4</i>
<br>
<br>
<br>

Sample Input:
```
2.3   →   a = 2.3
1.9   →   b = 1.9
```
Sample Output:
```
4
```
